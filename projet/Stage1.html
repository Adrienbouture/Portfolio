Projet de stage de première année



Contexte du projet

J'ai effectué ce stage de 4 semaines, sous la supervision de M. Dupont. Il souhaitait ameliorer sa plateforme "Projets Édouard Gand" qui présente les réalisations pédagogiques des étudiants.
Ma mission principale consistait à développer un panneau d'administration sécurisé permettant aux responsables pédagogiques de gérer eux-mêmes les projets présentés sur la plateforme.
Le projet impliquait la création d'interfaces de gestion (ajout, modification, suppression de projets) et l'intégration avec la base de données existante.
Cette expérience m'a permis de découvrir le PHP dans un contexte professionnel et de développer mes compétences en développement web.
Ensuite j'ai travaillé sur la plateforme www.portail-sio.fr qui est utilisée par les étudiants afin de pouvoir suivre leur progression lors de leur BTS ou Bachelor.



Technologies utilisées

HTML CSS PHP MySQL
