Projet de stage de deuxième année



Contexte du projet

Durant six semaines, j'ai réalisé mon stage de deuxième année avec un autre élève sous la supervision de M. Dupont.
Notre mission consistait à concevoir et développer en PHP une plateforme baptisée « SioConnect », destinée à centraliser et valoriser les données des anciens élèves du BTS SIO du lycée Édouard Gand.
L'objectif était de fournir aux futurs étudiants un outil leur permettant de découvrir des opportunités de stage et d'emploi via un site web interactif.



Technologies utilisées

HTML CSS JavaScript PHP MySQL



Mission réalisée

L'interface a été pensée pour offrir une expérience utilisateur fluide et intuitive : une page d'accueil présentant un aperçu global, un menu de catégorisation clair et des pages dédiées à chaque type d'information (parcours professionnel, scolaires, stages, poste actuel).
Chaque section permettait de filtrer, rechercher et consulter les profils des anciens élèves de manière structurée.

Page d'accueil de SioConnect

Dans le projet "SioConnect", j'ai également développé un espace de connexion pour permettre aux utilisateurs de créer un compte et de se connecter à leur profil. Cet espace de connexion offrait des fonctionnalités supplémentaires, tel que la gestion des informations personnelles.

Code du traitement des données de la connexion

Pour gérer leurs données, j'ai utilisé une base de données SQL. La modélisation de la base de données a été réalisée de manière à stocker les informations pertinentes pour chaque utilisateur, telles que le nom, la prenom, l'email, mais aussi leurs stages effectués, leurs poste actuel, etc. J'ai créé des tables pour les utilisateurs, les anciens elèves, les stages, et les postes actuels en établissant les relations nécessaires entre elles pour assurer une bonne intégrité des données.

Modèle Conceptuel des Données

La page "profil" a un visuel simple et intuitif afin que l'utilisateur puisse renseigner ces informations personnelles, son poste actuel, ou encore tous les stages qu'il a réalisé au cours de son parcours professionnel.

La page "Eleves" contient un tableau qui affiche l'ensemble des anciens èleves stocké dans la base de données.

Page de liste des élèves

Une barre de recherche est mise a disposition afin de rechercher une informations dans n'importe quelle colonne du tableau. Une fois la recherche effectuée, seulement les élèves ayant une données correspondant a ce critère apparaiteront dans le tableau.

Fonctionnalité de recherche d'élèves

Lorsque l'on clique sur un élève un pop up s'ouvre et affiche l'ensemble des données de cet élève.

Ensuite la page "Stage" fonctionne exactement de la meme façon que la page "Eleves" cependant le tableau affiche l'entreprise dans laquelle l'élève a éffectué son stage, l'email de son maitre de stage et la durée de ce dernier.
